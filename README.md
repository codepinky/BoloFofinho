# Bolo Fofinho — Entrega Aula 2

Três páginas do site com formulário de contato (HTML apenas, sem CSS).

**Desenvolvedor:** Marcos Librelon Raia  
**Disciplina:** Tecnologias Web — Mackenzie, 2026

## Conteúdo

- `index.html` — Homepage (com link para Contato)
- `sabores.html` — Página de sabores
- `contato.html` — Página de formulário com tabela para alinhamento dos campos
- Imagens e vídeos utilizados nas páginas

## Formulário

O formulário de contato possui 6 campos de tipos diferentes:
- Nome (text)
- Email (email)
- Telefone (tel)
- Sabor do bolo (select)
- Data desejada (date)
- Mensagem (textarea)

Utiliza tabela para alinhamento dos campos e atributos HTML5 (required, placeholder, pattern, maxlength).

## Como executar

Abra o arquivo `index.html` em um navegador.
