# Bolo Fofinho — Entrega Aula 1

Duas páginas iniciais do site (HTML apenas).

**Desenvolvedor:** Marcos Librelon Raia  
**Disciplina:** Tecnologias Web — Mackenzie, 2026  
**Tema:** Vendas Online (Comida)

## Conteúdo

- `index.html` — Homepage com textos, listas, links, imagens e vídeo
- `contato.html` — Página de contato com link "Fale Conosco"
- Navegação de ida e volta entre as duas páginas
- Pasta `images/` — Imagens utilizadas
- Pasta `videos/` — Vídeos utilizados

## Elementos HTML utilizados

- Textos (títulos, parágrafos)
- Listas (ul, ol)
- Links (navegação e Fale Conosco)
- Imagens
- Vídeo

## Como executar

Abra o arquivo `index.html` em um navegador.
